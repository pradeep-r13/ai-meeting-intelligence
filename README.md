# AI Meeting Intelligence & Decision Tracker

An enterprise-grade GenAI application that transforms meeting transcripts into actionable intelligence.

## Features

- Decision Extraction
- Action Item Tracking
- Risk Analysis
- Meeting Summarization
- RAG-based Meeting Search
- Cross-Meeting Intelligence
- Contradiction Detection
- Follow-up Generation

## Tech Stack

- Streamlit
- LangGraph
- Ollama
- Qwen 3
- Qdrant
- SQLite
- Python

## Architecture

Meeting Transcript
→ Agents
→ Structured Outputs
→ Vector Database
→ RAG
→ Analytics Dashboard
